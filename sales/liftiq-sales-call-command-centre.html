Follow-up generator

Send short, grounded follow-up.

Templates are intentionally plain. No password values, no fake urgency, no public pricing unless the call already discussed it.

Portal access wording

For test portal invite emails, include the login email only if approved. State that the temporary password is supplied separately and first login forces password change. Never include a password value in this artifact or the same email thread.

Do-not-claim guardrails

Stay honest when the prospect pushes.

Boundary language is not weakness. It is how the pilot earns trust.

Do not claim

  • Compliance approval
  • Permit approval
  • Legal road access
  • WHS guarantee
  • Engineered lift confirmation
  • Autonomous dispatch
  • Safe to dispatch
  • Payroll-ready
  • Full ERP replacement
  • Fleet Cost & Care replacement today

Approved framing

  • Decision support
  • Review required
  • Dispatcher remains decision-maker
  • Before-dispatch clarity
  • Explainable recommendations

  • Audit trail
  • Pilot-stage
  • Managed pilot
