# DispatchTalon LinkedIn Ad Variant Matrix

## Creative Strategy

The pack uses four distinct category-entry points:

| Ad | Category-entry point | Buyer thought to trigger |
|---|---|---|
| Job Intake | Rough job notes | The dispatch problem starts before allocation. |
| SmartRank | Allocation reasoning | I need to see why the recommendation was made. |
| Operating Modes | Fit to company model | This can work for labour-only or plant + labour operations. |
| Crane Transport Review | Crane/counterweight assumptions | Planning assumptions need a visible review trail. |

The creative system is intentionally simple:
- one problem per image
- one proof-style visual per image
- one CTA
- no broad product manifesto
- no generic SaaS language

## Image Selection Rationale

| Ad | Image choice | Rationale |
|---|---|---|
| Ad 1 | Public DispatchTalon screenshot plus Job Brief Import preview panel | No authenticated Job Brief Import screenshot was available. The panel is labelled as a pilot workflow preview and uses sanitised synthetic job details. |
| Ad 2 | SmartRank preview panel | SmartRank needs a close, readable proof visual. Existing authenticated screenshot was unavailable, so the panel shows sanitised synthetic ranking logic. |
| Ad 3 | Public OG artifact plus Our Business preview panel | Operating mode is a current product truth and is best shown as two explicit mode cards. |
| Ad 4 | Crane/transport review preview panel | The message is specific enough that a generic public screenshot would be too weak. The panel is labelled as a pilot workflow preview. |

## Targeting Notes

Primary LinkedIn job functions:
- Operations
- Logistics
- Program and Project Management
- Construction
- Engineering

Primary job titles:
- Operations Manager
- Dispatch Manager
- Crane Supervisor
- Lift Manager
- Transport Manager
- Allocations Manager
- General Manager
- Managing Director
- Owner
- Safety Manager

Company types:
- Crane hire
- Rigging
- Heavy lift
- Labour hire
- Shutdown labour
- Plant hire with operators
- Transport-supported lifting
- Industrial services

Company size hypothesis:
- 11 to 50: strongest owner/operator message
- 51 to 200: strongest dispatch manager and operations manager message
- 201 to 500: strongest audit/process/proof message

Geography:
- Australia first
- Queensland and East Coast tests first if budget is constrained
