## Preferred Product References

Use:

- DispatchTalon.
- DispatchTalon by Pressure Systems.
- Pressure Systems DispatchTalon where useful.
- DispatchTalon portal.
- DispatchTalon pilot.
- DispatchTalon Sales Call Command Centre.

Use sparingly:

- DispatchTalon, previously developed as LIFTIQ.
- DispatchTalon is the new working product name for the dispatch intelligence platform previously developed as LIFTIQ.

Do not use:

- Dispatchtalon.
- Dispatch Talon.
- Dispatch-Talon.
- LIFTIQ as a current public product name.
- LiftIQ as a current public product name.

## Technical Identifiers Retained For Now

Retained temporarily:

- `/liftiq/` public route.
- `https://liftiq-pilot.fly.dev/console/`.
- Fly app name `liftiq-pilot`.
- Database filename/defaults using `liftiq.db`.
- Existing local storage keys such as `liftiq.token`.
- Environment variables with `LIFTIQ_` prefixes.
- Source folders such as `content/liftiq-linkedin-ads/`.
- CSS/JS filenames such as `assets/css/liftiq.css` and `assets/js/liftiq.js`.

These are not current public product-name endorsements. They are retained to prevent route, deployment, or automation breakage.
