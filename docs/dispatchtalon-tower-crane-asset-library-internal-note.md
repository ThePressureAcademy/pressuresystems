# Tower Crane Asset Library - Static Internal Reference Feature

Status: internal product note

Product: DispatchTalon by Pressure Systems

Scope: static/internal console reference feature

## Purpose

The Tower Crane Asset Library is a static internal reference asset library inside the DispatchTalon console. It supports business-building conversations, customer-facing page planning, crane-category organisation, and pilot-user education.

It is not a verified fleet database.

## Governance Boundary

- The listed crane models must not imply that any pilot user, competitor, prospect, or reference company owns the equipment.
- The listed crane models must not imply active hire availability, partner access, brokered access, exclusivity, or customer-ready supply.
- All crane models remain verification candidates unless specifically confirmed with source evidence for the relevant business.
- Spec links must remain withheld unless the specific file or technical page is verified and appropriate for customer-facing use.
- Customer-facing output should use neutral language such as reference asset, candidate model, equipment category example, verify before publishing, and specs available on request.

## Out Of Scope

- Backend persistence.
- Database schema changes.
- Tenant-editable workflows.
- File upload or spec-sheet management.
- Public website routes.
- Real spec links.
- Company-specific fleet claims.

## Expansion Rule

Do not expand this into a persisted asset database, editable product module, upload workflow, or public website generator unless pilot-user demand proves the workflow is needed.
